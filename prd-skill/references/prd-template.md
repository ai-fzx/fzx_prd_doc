# PRD 完整模板

可直接复制使用的完整 PRD 文档模板。

---

```markdown
# PRD：[产品/功能名称]

## 背景与问题
[要解决什么问题、为谁、当前痛点或机会]

## 目标与成功指标
- **目标**：[业务/产品目标]
- **成功指标**：
  - 指标1：[目标值]（当前基线：[当前值]）
  - 指标2：[目标值]（当前基线：[当前值]）

## 用户与场景
- **目标用户**：[具体角色 + 背景]
- **核心场景**：
  1. [场景1：从...到...]
  2. [场景2：从...到...]
  3. [场景3：从...到...]

## 功能需求

| ID | 用户故事 | 优先级 |
|----|---------|--------|
| R1 | 作为 [角色]，我希望 [能力]，以便 [价值] | P0 |
| R2 | 作为 [角色]，我希望 [能力]，以便 [价值] | P1 |
| R3 | 作为 [角色]，我希望 [能力]，以便 [价值] | P2 |

## 范围与边界
- **In scope**：本期实现范围
- **Out of scope**：本期明确不做

## 验收标准

| 需求ID | 验收条件 |
|--------|---------|
| R1 | 当 [前置条件] 时，系统应 [预期行为] |
| R2 | 当 [前置条件] 时，系统应 [预期行为] |

## 依赖与约束
- **技术依赖**：[依赖方 / 依赖系统]
- **时间约束**：[上线时间节点]
- **合规约束**：[适用法规或政策]
- **性能约束**：[性能指标要求]

## 非目标
本期明确不解决的问题：
1. [问题1]
2. [问题2]

## 附录
- **术语表**：[术语定义]
- **变更记录**：日期 - 变更内容 - 变更人
```
